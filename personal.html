Projects


Slot Machine
This C++ program simulates a slot machine. In the program, you can add money to your account, place your bets, and play a simulated version of a slot machine which uses ASCII characters to display the icons typically used in slot machines.

Image Manipulator
This Java program takes an image and manipulates the pixels to attain certain effects. These effects include shifting the image in 4 different directions, flipping the image horizontally and vertically, expanding the image and changing the RGBA properties of the image.
